  title: Freezino API
  description: |
    Educational casino simulator API for Freezino application.

    **WARNING**: This application uses NO real money. All currency is virtual and for educational purposes only.

    ## Authentication
    - Local authentication (username/password)
    - Google OAuth 2.0 for user authentication
    - JWT tokens for API authorization

    ### How to authenticate
    1. Register via `POST /auth/register` or login via `POST /auth/login` (or use Google OAuth)
    2. You will receive `access_token` and `refresh_token`
    3. For protected endpoints (marked with 🔒), add header:
       ```
       Authorization: Bearer <access_token>
       ```
    4. When access token expires, use `POST /auth/refresh` with refresh_token to get a new one

    ## Features
    - Work system with multiple job types
    - Casino games (Roulette, Slots, Blackjack, Crash, Hi-Lo, Wheel)
    - Virtual item shop
    - Loan system (friends, bank, microcredit)
    - Real-time blackjack via WebSocket

    ## Base URL
    - Development: `http://localhost:3000/api`
    - WebSocket: `ws://localhost:3000/ws`
    - Production: Update with your production URL
